# MIT African Futures

## [FR] Couleurs

Le système se base sur 5 couleurs 

- `green`
- `red`
- `yellow`
- `purple`
- `orange`

### Bannière (titre + résumé)

Pour ajouter un fond de couleur à la bannière en haut de page (fond blanc par défaut), il faut éditer la page sur le back office, puis remplire le champ "Classe CSS spécifique" (à droite du titre) avec une des couleurs ci-dessus.

### Blocs chapitre et appels à actions

Pour ajouter un fond de couleur à un de ces deux blocs, il faut 2 choses :

- Utiliser la mise en forme "forte mise en avant";
- Cliquer sur "Réglages avancés" (le rouage) tout en bas à gauche et ajouter la couleur ci-dessus dans le champ "classe spécifique".

## [EN] Colors

The system is based on 5 colours:

- `green`
- `red`
- `yellow`
- `purple`
- `orange`

### Banner (title + summary)

To add a coloured background to the banner at the top of the page (white background by default), edit the page in the back office, then fill in the ‘Specific CSS class’ field (to the right of the title) with one of the colours above.

### Blocks: chapter and calls to action

To add a coloured background to one of these two blocks, you need to do two things:

- Use the ‘strong emphasis’ formatting;
- Click on ‘Advanced settings’ (the cogwheel) at the bottom left and add the colour above in the ‘specific class’ field.
