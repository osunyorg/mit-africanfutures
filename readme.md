# MIT African Futures

## Couleurs

Le système se base sur 5 couleurs 

- `green`
- `red`
- `yellow`
- `purple`
- `orange`

### Bannière (titre + résumé)

Pour ajouter un fond de couleur à la bannière en haut de page (fond blanc par défaut), il faut éditer la page sur le back office, puis remplire le champ "Classe CSS spécifique" (à droite du titre) avec une des couleurs ci-dessus.

### Blocs chapitre et appels à actions

Pour ajouter un fond de couleur à un de ces deux blocs, il faut 2 choses :

- Utiliser la mise en forme "forte mise en avant";
- Cliquer sur "Réglages avancés" (le rouage) tout en bas à gauche et ajouter la couleur ci-dessus dans le champ "classe spécifique".
